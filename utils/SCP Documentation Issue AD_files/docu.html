Instruction Set

Keywords of SCP are - like regular XML - case-sensitive.

The indication of a feature's "availability" refers to the minimum version of a component required for the respective feature. Abbreviations for components are as follows:

  • TC: TaskControl
  • IG: ImageGenerator
  • TR: Traffic
  • MM: ModuleManager

Two components are meanwhile obsolete. They have been replaced by the moduleManager, so wherever the following components are mentioned, it applies to the moduleManager:

  • SM: SensorManager
  • DM: DynamicsManager

If no component is given in the last column of the table, then the feature is foreseen to be implemented in an upcoming version of the software package.

A distinction is made whether a command can only be used in the respective component's configuration file (C) or whether it may be used during run-time (R) also by sending it via network. The standard configuration files of the components are:

  • TC: taskControl.xml
  • MM: moduleManager.xml
