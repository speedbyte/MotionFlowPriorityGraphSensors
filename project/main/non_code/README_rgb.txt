Object Detection and Multi-Object Tracking Ground Truth
--------------------------------------------------------

Link: http://download.xrce.xerox.com/virtual-kitti-1.3.1/vkitti_1.3.1_rgb.tar

Each video is simply a folder in the format:

    vkitti_<version>_rgb/<world>/<variation>/%05d.png

PNG files are encoded in RGB with 8-bit per channel.
